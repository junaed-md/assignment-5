Question-1: What are the different ways to select an element in the DOM?

There are a few ways to select elements in the DOM. Example: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is actual HTML code, innerText is the visible text and textContent is all text content including hidden elements.

Question-3: What is event delegation in the DOM?

A parent is connected to a single listener through event delegation. It manages events from bubble-up descendants.

Question-4: What is event bubbling in the DOM?

A DOM event propagation model known as "event bubbling" describes how an event on a target element moves up the DOM tree to its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tag'). Add: parentNode.appendChild(childNode). Remove: parentNode.removeChild(childNode) or childNode.remove()
